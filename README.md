# ConDew留学指导网站

## 项目概述

这是一个轻量级的留学指导公司网站，使用纯HTML + CSS + JavaScript开发，具有以下特点：

- 响应式设计，支持移动端和桌面端
- 轮播图展示
- 学习资料展示（链接到闲鱼）
- 留言板功能
- 可复用的导航组件

## 文件结构

```
web/
├── index.html              # 首页
├── resources.html          # 学习资料页面
├── message-board.html      # 留言板页面
├── about.html              # 关于我们页面
├── contact.html            # 联系我们页面
├── css/
│   └── style.css          # 主样式文件
├── js/
│   ├── navigation.js      # 导航组件（可复用）
│   ├── carousel.js        # 轮播图功能
│   └── message-board.js   # 留言板功能
├── images/                # 图片文件夹
└── README.md              # 说明文档
```

## 如何添加新的学习资料

在 `resources.html` 文件中，找到对应的分类部分，按照以下格式添加新的资料卡片：

```html
<div class="card resource-card">
    <div class="price">¥价格</div>
    <div class="card-image">📚</div>
    <h3>资料标题</h3>
    <p>资料描述</p>
    <a href="您的闲鱼链接" target="_blank" class="btn">前往购买</a>
</div>
```

### AP笔记系列特殊样式
AP笔记系列使用了特殊的CSS类 `ap-notes-section`，具有以下特效：
- 顶部渐变线条
- 悬停时的左边框高亮
- 价格标签脉冲动画
- 卡片悬停时的上升效果

## 如何修改闲鱼链接

1. 找到对应的资料卡片
2. 修改 `<a href="...">` 中的链接地址
3. 确保添加 `target="_blank"` 属性以在新窗口打开

## 部署说明

这是一个纯静态网站，可以部署到任何支持静态文件的服务器：

1. 将整个 `web` 文件夹上传到服务器
2. 确保服务器支持HTML、CSS、JavaScript文件
3. 访问 `index.html` 即可

## 本地预览

使用Python启动本地服务器：

```bash
cd web
python3 -m http.server 8080
```

然后在浏览器中访问 `http://localhost:8080`
